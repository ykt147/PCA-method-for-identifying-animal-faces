# 基于 PCA 的狮子与老虎图像识别

利用主成分分析（PCA）方法对狮子和老虎的图像进行降维与分类识别。PCA 是一种以"最小重构误差"为目标的线性降维方法，通过提取数据中最重要的信息来压缩数据存储空间，并用于后续的特征分析与数据重构。

## 项目特性

- **平均脸提取**：计算并可视化狮子与老虎的平均脸
- **特征分析**：输出协方差矩阵的特征值与特征向量，展示特征值衰减趋势与累积方差贡献率
- **自动参数搜索**：通过网格搜索优化主成分数和 KNN 邻居数，实现高准确率分类
- **降维可视化**：将高维图像数据映射到二维空间，直观展示类别分布

## 方法说明

### 数据处理流程

1. **图像加载**：读取 `dataset/lion` 和 `dataset/tiger` 中的图片，统一转为灰度图并缩放至 64×64 像素，展平为 4096 维向量
2. **数据划分**：前 2/3 作为训练集，后 1/3 作为测试集
3. **PCA 降维**：对训练数据进行 PCA 变换，自动搜索最优主成分数
4. **标准化**：对降维后的数据进行 StandardScaler 标准化处理
5. **KNN 分类**：使用 K 近邻分类器对测试集进行识别

### 参数优化

程序通过 `find_optimal_solution` 和 `optimize_k_and_components` 函数自动搜索最优参数组合：

- **主成分数**：从 1 到 min(训练样本数-1, 特征维度, 50) 遍历
- **KNN 邻居数 k**：遍历不超过 min(2N+1, 40) 的值
- **预处理方式**：对比"原始"和"标准化"两种预处理策略

当找到满足准确率要求的方案时提前返回，否则返回最优结果。

### 可视化输出

程序运行后会弹出一个 3×3 的可视化面板，包含：

- **平均脸**：训练集图像的像素均值
- **特征脸（Eigenfaces）**：前两个主成分对应的图像
- **特征值柱状图**：前 20 个特征值的大小对比
- **特征值衰减曲线**：展示特征值随索引的衰减趋势
- **测试数据散点图**：前两维主成分空间中的测试样本分布
- **累积方差贡献率**：反映主成分数量与信息保留比例的关系
- **分类统计与详细预测结果**

## PCA 与 SVD 方法的差异讨论

PCA 和 SVD（奇异值分解）均可用于数据降维，但在解决此类图像识别问题时有以下差异：

### 1. 数学原理

- **PCA**：通过对数据的协方差矩阵 $X^T X$ 进行特征值分解，取最大特征值对应的特征向量作为主成分方向。
- **SVD**：直接对数据矩阵 $X$ 进行奇异值分解 $X = U \Sigma V^T$，右奇异向量 $V$ 即为 PCA 的主成分方向，奇异值的平方 $\sigma_i^2$ 与协方差矩阵的特征值成正比。

### 2. 计算效率

- 当特征维度 $d$ 远大于样本数 $n$ 时（如本例中 $d=4096, n=40$），PCA 需要计算 $4096 \times 4096$ 的协方差矩阵，计算量和存储开销巨大。而 SVD 可以对 $n \times d$ 的数据矩阵直接分解，实际计算中通常更高效。
- 在现代数值计算库（如 NumPy、scikit-learn）中，PCA 的内部实现通常也调用 SVD，而非显式构造协方差矩阵。

### 3. 数值稳定性

- SVD 避免了显式计算 $X^T X$，从而避免了协方差矩阵计算中可能出现的数值精度损失（条件数平方效应）。SVD 的数值稳定性优于直接的特征值分解方法。

### 4. 实际使用

- 在 scikit-learn 中，`PCA` 默认使用 SVD 求解器（`svd_solver='auto'`），因此两种方法在实际调用时底层是统一的。
- 若手动实现，使用 SVD 的代码更简洁：`U, S, Vt = np.linalg.svd(X_centered, full_matrices=False)`，取 `Vt` 的前 $k$ 行即为投影方向。

### 5. 总结

| 对比维度 | PCA（特征值分解） | SVD |
|---------|------------------|-----|
| 分解对象 | 协方差矩阵 $X^T X$ | 数据矩阵 $X$ |
| 高维效率 | $d \gg n$ 时效率低 | 更高效 |
| 数值稳定性 | 较差（条件数平方） | 更好 |
| 结果等价性 | 主成分方向相同 | 主成分方向相同 |

对于本项目的图像分类任务，两种方法在理论上等价，但 SVD 在实现效率和数值稳定性上更具优势。
